# Crypto Price Tracker

A JavaScript project that fetches and displays live cryptocurrency prices using the CoinGecko API.

## Features
- Fetch live crypto prices
- Display cryptocurrency images
- Responsive card layout
- Dynamic DOM rendering
- Error handling

## Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API
- CoinGecko API
